# קמפיינר 10X — סקילים וסוכנים לקלוד קוד

סקילים וסוכנים AI לשימוש עם Claude Code בסדנת קמפיינר 10X.

## התקנה

```bash
git clone https://github.com/asafyakov/campainer-10x-skills.git
cd campainer-10x-skills
bash install.sh
```

פותחים Claude Code חדש — הסקילים זמינים.

## מבנה

```
├── מפגש-1/              ← הקמת סוכנות + חיבור META
├── מפגש-2/              ← ניהול לקוחות + קמפיין מקצה לקצה
├── agents/              ← סוכני AI אוטומטיים
└── install.sh           ← מתקין הכל
```
