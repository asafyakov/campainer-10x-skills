import sys, os

- קיים → תמלל אוטומטית
- לא קיים → הדרך:

> **צריך חיבור ל-Replicate (חד-פעמי, כמעט חינם):**
> 1. נכנסים ל-replicate.com ונרשמים (חינם, אפשר עם גוגל/גיטהאב)
> 2. נכנסים ל-Account Settings → API tokens
> 3. לוחצים "Create token" ומעתיקים
> 4. אומרים לקלוד קוד: "תוסיף ל-~/.claude/master_keys.py את המפתח: `REPLICATE_API_TOKEN = 'r8_...'`"
>
> **עלות:** קרדיט חינמי לחשבון חדש. אחרי זה כ-3 שקלים ל-100 דקות הקלטה.

**תמלול:**

```python
import replicate

def transcribe_audio(file_path):
    output = replicate.run(
        "openai/whisper:cdd97b257f93cb89dede1c7584df3920f8b02b90702d2a04f1e229d7508e5f99",
        input={
            "audio": open(file_path, "rb"),
            "model": "large-v3",
            "language": "he",
            "translate": False
        }
    )
    return output.get("transcription", "")
```

## עיבוד ופלט

קרא את קובץ [template.md](template.md) — זה הפורמט המדויק שצריך למלא.

**כללי עיבוד — חובה:**

1. **ציטוטים מדויקים** — השתמש בלשון המשתמש/תמלול מילה במילה בכל מקום שאפשר. אסור לפרפרז.
2. **תבניות חוזרות** — אם 3 לקוחות אמרו דבר דומה, זה נכנס כעובדה מוצקה.
3. **סצנות, לא תחושות** — "מרגיש טוב" = לא מספיק. "קם בבוקר בלי לבדוק כמה לידים יש לו" = מספיק.
4. **שפת הלקוח** — ציטוטים מילה במילה בלבד. 4 קטגוריות: כאב / תסכול / חלום / טריגר.
5. **חסר מידע** — אם שאלה ספציפית לא כוסתה, שאל שאלת המשך אחת ממוקדת. אל תמציא.
6. **עברית תמיד** — הפלט כולו בעברית.

**לאחר העיבוד — הוצא את המסמך המלא לפי template.md.**

## Pre-Delivery Check

לפני שאתה מגיש את המסמך, בדוק:

1. **ספציפיות** — האם יש לפחות סצנה מוחשית אחת בכל סקשן? אם יש "מרגיש לחץ" בלי סיטואציה — תוסיף.
2. **ציטוטים** — האם יש לפחות 2 ציטוטים מדויקים בסקשן "שפת הלקוח"? אם לא — חזור לחומר.
3. **היום אחרי** — האם Have/Feel/Average Day/Status מולאו בסצנות ולא בתחושות?
4. **קונפליקט מרכזי** — האם הניסוח הוא "[מה שהוא רוצה] בלי [מה שהוא מפחד]"?

אם משהו חסר — השלם לפני שאתה מגיש.
