# התקנת סקילים וסוכנים — קמפיינר 10X מפגש 2

כשקלוד קוד קורא את הקובץ הזה — עליו לבצע את ההתקנה אוטומטית:

## הוראות התקנה

1. העתק את כל תיקיות הסקילים מ-`skills/` ל-`~/.claude/skills/`
2. העתק את כל קבצי הסוכנים מ-`agents/` ל-`~/.claude/agents/`
3. הדפס אישור לכל סקיל וסוכן שהותקן
4. בסוף — הדפס רשימת כל מה שהותקן

## מה מותקן

### סקילים (`~/.claude/skills/`)
- `new-customer` — הגדרת לקוח חדש, יוצר CLAUDE.md מלא
- `dream-customer` — בניית פרופיל אווטאר מלא (avatar.md) — מהקלטות או מ-9 שאלות
- `facebook-ad-copywriter` — כתיבת קופי מלא למודעות פייסבוק
- `competitor-research` — מחקר מתחרים מ-Facebook Ads Library
- `creative-ai` — יצירת קריאייטיבים עם KIE AI
- `facebook-campaign` — העלאת קמפיין דרך Facebook API
- `skill-creator` — יצירת סקילים חדשים (רשמי של Anthropic)

### סוכנים (`~/.claude/agents/`)
- `copy-reviewer` — ביקורת קופי לפי Seven Sweeps
- `qa-campaign` — בדיקת קמפיין לפני העלאה
