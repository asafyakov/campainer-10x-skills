# Dashboard Sync — חיבור CRM לדשבורד הסוכנות

> סקיל Claude Code שמסנכרן לידים מ-CRM (רב מסר / גוגל שיטס / מאנדיי) לדשבורד הסוכנות.
> **פעם ראשונה:** setup של 2 דקות. **מהפעם השנייה:** Claude Code מסנכרן הכל אוטומטית בכל פתיחה.

---

## התקנה — 3 צעדים

### צעד 1 — הורד את הסקיל

```bash
# גיט של הסדנה כבר מורד? הסקיל כבר אצלך.
# אם לא — הורד את תיקיית dashboard-sync/ לשולחן העבודה.
```

### צעד 2 — התקן

פתח טרמינל, נווט לתיקייה, הרץ:

```bash
bash install.sh
```

זה מעתיק את הסקיל למקום הנכון (`~/.claude/skills/dashboard-sync/`).

### צעד 3 — הגדר (פעם אחת בלבד)

פתח Claude Code **בתיקיית הלקוחות שלך** וכתוב:

```
/dashboard-sync setup
```

Claude Code ישאל אותך **שאלה אחת בלבד** — ה-API Key מהדשבורד — ויטפל בכל השאר לבד:
- שומר את ה-Key
- מעדכן את ה-CLAUDE.md שלך
- בודק שהחיבור עובד
- מסיים עם ✅

**זהו. לא צריך לעשות כלום יותר.**

---

## שימוש יומיומי

| פקודה | מה קורה |
|--------|----------|
| פותח Claude Code בתיקיית הלקוחות | סנכרון אוטומטי לכל הלקוחות |
| `/dashboard-sync all` | סנכרן הכל ידנית |
| `/dashboard-sync מילי וליהי` | סנכרן לקוח ספציפי |
| `/dashboard-sync status` | ראה מתי סונכרן כל לקוח |

## הוספת לקוח חדש

לכל לקוח שרוצים שהדשבורד יראה לידים שלו — מוסיפים ל-`client.json` שלו:

**רב מסר:**
```json
"crm_sync": {
  "type": "rav_mesar",
  "list_id": 103567,
  "ref_field": "ref"
},
"connections": {
  "rav_mesar": {
    "client_id": 231,
    "client_secret": "...",
    "user_token": "...",
    "account_id": 1005807
  }
}
```

**גוגל שיטס:**
```json
"crm_sync": {
  "type": "google_sheets",
  "sheet_id": "1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms",
  "ref_field": "utm_campaign"
}
```

## מה ה-API Key?

מפתח ייחודי לכל משווק — מאמת שהלידים שאתה דוחף הולכים לחשבון שלך בדשבורד ולא של אחר.

**לקבל:**
1. נכנס לדשבורד → הגדרות → מפתחות API
2. לוחץ "צור מפתח חדש"
3. נותן שם (למשל "Claude Code — מחשב עבודה")
4. מעתיק — מוצג פעם אחת בלבד

## שאלות נפוצות

**Q: האם הלידים של לקוח A יכולים לדלוף ללקוח B?**
לא. כל API Key מקושר למשתמש ספציפי — הלידים תמיד נכנסים לחשבון הנכון.

**Q: מה קורה אם ליד כבר קיים בדשבורד?**
הסקיל עושה upsert — מעדכן אם קיים, מוסיף אם חדש. לא יהיו כפילויות.

**Q: האם צריך לסנכרן כל פעם ידנית?**
לא — אחרי ה-setup, בכל פתיחת Claude Code בתיקיית הלקוחות הסנכרון רץ אוטומטית.
