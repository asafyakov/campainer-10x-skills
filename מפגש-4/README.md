# מפגש 4 — אוטומציה עם Claude Code

ברוך הבא למפגש האחרון של קמפיינר 10X.

במפגש הזה תתקין כלים שעובדים בשבילך **גם כשאתה לא ליד המחשב** — בוט טלגרם, בוט WhatsApp, וסוכן מחקר מתחרים אוטומטי.

---

## מה יש פה

### 🤖 בוט טלגרם — `telegram-bot-skill/`
שלח הודעה מהטלפון → Claude מעבד → תשובה חוזרת אליך.

**מתי להשתמש:** אם אתה רוצה לדבר עם Claude מהטלפון דרך Telegram  
**זמן התקנה:** ~5 דקות  
**דרישות:** Python 3.9+, חשבון Telegram

```
הדבק בקלוד קוד:
התקן לי את סקיל הטלגרם לפי הקובץ telegram-bot-skill/SKILL.md
```

---

### 💬 בוט WhatsApp — `whatsapp-bot-skill/`
אותו רעיון — רק דרך WhatsApp. שני מצבים: אישי (גישה מלאה לקבצים) או צ'אטבוט (לקבוצות).

**מתי להשתמש:** אם אתה רוצה לדבר עם Claude דרך WhatsApp  
**זמן התקנה:** ~10 דקות  
**דרישות:** Node.js 18+, Claude Code מותקן

```
הדבק בקלוד קוד:
התקן לי את סקיל ה-WhatsApp לפי הקובץ whatsapp-bot-skill/SKILL.md
```

---

### 🔍 מחקר מתחרים אוטומטי — `competitor-research/`
סורק פייסבוק, אינסטגרם, גוגל — מזהה מתחרים מובילים, מוריד קריאייטיבים, מנתח הוקים ודפי נחיתה, ומייצר דוח מלא.

**מתי להשתמש:** לפני כל קמפיין חדש, לכל לקוח חדש  
**זמן ריצה:** 25–45 דקות  
**דרישות:** Apify token (חינם להתחלה)

```
הדבק בקלוד קוד:
תריץ מחקר מתחרים לפי הקובץ competitor-research/SKILL.md
הנישה: [כתוב כאן את הנישה של הלקוח שלך]
```
